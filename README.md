## Topics Covered

### Cloud & Infrastructure
- Cloud computing fundamentals (IaaS, PaaS, SaaS, FaaS, deployment models)
- **AWS**: Compute, Storage, Database, Networking, Security, AI/ML, DevOps — 90+ services
- **Azure**: App Services, Cosmos DB, Synapse, OpenAI, Entra ID — 85+ services
- **GCP**: BigQuery, Spanner, Vertex AI, Cloud Run, Anthos — 80+ services

- Core fundamentals: networking, APIs, databases, caching
- Scalability, availability, and performance trade-offs
- Architecture patterns: Microservices, SOA, Monolith, CQRS, Saga
- Data strategies: CAP theorem, consistency models, ACID, distributed transactions
- Caching strategies and cache invalidation techniques
- Load balancing and circuit breaker patterns
- REST vs GraphQL, hashing techniques

### Backend Engineering
- .NET / C# / ASP.NET Core / VB.NET
- Async/await, dependency injection, middleware
- JavaScript and React (Hooks, component patterns)
- Real-world migration experience: SSAS → AWS Redshift

### Interview Preparation
- STAR behavioral stories aligned to real work history
- Technical deep-dives mapped to job requirements
- Counter-questions and honest, resume-backed answers
- Confidence-rated study plans for structured last-minute prep

Notes here aim to go beyond surface-level definitions — they capture *when to use* something, *why it exists*, and *how it compares* to alternatives. Real examples from production experience are woven in wherever possible.
